    int num;
    printf("Enter a number:");
    scanf("%d", &num);
    printf("%d", num);

7-指针与函数与数组

指针和数组

指针在数组中特别有用。

声明一个数组后就为其元素保留了一个连续的内存地址块。通过指针，可以指向数组第一个元素，然后使用地址运算来遍历数组。

  • +向前移动到一个内存位置
  • -向后移动到一个内存位置

请看下面的程序：

#include <stdio.h>

int main() {
    int a[] = {22, 33, 44, 55, 66};
    int *ptr = NULL;

    ptr = a;
    for (int i = 0; i < 5; i++) {
        printf("%d ", *(ptr + i));
    }
    printf("\n");
    return 0;
}
